# 孙宇祥教授个人简历网站 - 产品需求文档

## 1. 产品概述

青年学者孙宇祥的个人学术简历网站，展示其从北师大数学本科到清华博导的卓越学术历程。面向学术同行、高校学生及工业界研究人员，传达专业、严谨且富有学术气息的个人形象。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 访问权限 |
|------|----------|
| 访客 | 浏览全部公开学术履历信息 |

### 2.2 功能模块
1. **首页**：学术履历时间线、教育背景、科研方向、学术成就
2. **个人介绍区**：照片、姓名、职位、机构、联系方式
3. **教育经历模块**：北师大数学本科、清华电子信息硕士、UCB计算机博士、MIT博士后
4. **科研方向模块**：大模型、人工智能、计算机科学
5. **学术成就模块**：发表论文、科研项目、获奖情况
6. **联系方式模块**：邮箱、办公地点

### 2.3 页面详情
| 页面 | 模块 | 功能描述 |
|------|------|----------|
| 首页 | Hero区域 | 展示姓名、职位、头像、教育背景概览 |
| 首页 | 教育经历时间线 | 可视化展示北师大→清华→UCB→MIT→清华的学术历程 |
| 首页 | 研究方向 | 大模型、人工智能、计算机科学等研究方向 |
| 首页 | 学术成就 | 论文发表、科研项目、荣誉奖项 |
| 首页 | 联系方式 | 邮箱、办公地点等联系信息 |

## 3. 核心流程

访客访问网站 → 浏览学术履历 → 查看教育经历时间线 → 了解研究方向 → 查看联系方式

```
用户访问 → 首页加载 → 滚动浏览各模块 → 查看联系方式或发送邮件
```

## 4. 用户界面设计

### 4.1 设计风格
- **风格定位**：学术简约风格，融合中国传统文化元素与国际化现代设计
- **色调**：深蓝色为主色调(#1a365d)，配合金色点缀(#d4af37)，传达学术严谨与卓越
- **字体**：标题使用思源宋体或衬线体，正文使用无衬线体
- **布局**：单页式设计，各模块垂直排列，平滑滚动导航

### 4.2 页面设计概览
| 页面 | 模块 | 视觉元素 |
|------|------|----------|
| 首页 | Hero | 全屏背景，渐变色，动态文字入场动画 |
| 首页 | 教育时间线 | 垂直时间线，节点动画，交替左右布局 |
| 首页 | 研究方向 | 卡片式布局，悬浮效果，图标装饰 |
| 首页 | 学术成就 | 数字统计动画，里程碑标记 |
| 首页 | 联系方式 | 简洁表单风格，社交图标 |

### 4.3 响应式设计
- 桌面端：多列布局，完整动画效果
- 平板端：自适应列数，保持核心布局
- 移动端：单列布局，优化触摸交互

### 4.4 动效设计
- 页面加载：元素依次淡入（stagger 100ms）
- 滚动触发：各模块进入视口时触发入场动画
- 悬浮交互：卡片悬浮上浮阴影效果
- 时间线：节点依次点亮动画
